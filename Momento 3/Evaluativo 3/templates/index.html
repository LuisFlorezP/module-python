Informe de Análisis de Datos

Obtención de Datos

Utilizamos la API de GitHub para obtener información sobre repositorios públicos con más de 10,000 estrellas. La API proporcionó detalles sobre varios repositorios, incluyendo el número de estrellas, forks, y issues abiertos.

Análisis de Datos

Los datos obtenidos se cargaron en un DataFrame de Pandas. Exploramos los datos utilizando funciones como head(), info() y describe() para comprender su estructura y contenido.

Preprocesamiento de Datos

Se seleccionaron las columnas relevantes y se eliminaron filas con valores nulos. También se convirtieron los tipos de datos a formatos apropiados.

Conclusiones

Los gráficos muestran que los repositorios más populares en GitHub (por número de estrellas) incluyen proyectos muy conocidos como FreeCodeCamp y Vue.js. Además, hay una correlación positiva entre el número de estrellas y el número de forks, lo que sugiere que los proyectos más populares también tienden a ser los más copiados.
